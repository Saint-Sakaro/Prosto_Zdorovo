# ✅ Форма отзыва упрощена - ФИНАЛЬНАЯ ВЕРСИЯ

## Что изменено:

### 1. Скрыты поля, которые предзаполнены из POI:
- ✅ **Координаты** (широта/долгота) - скрыты, если переданы из POI
- ✅ **Категория** - скрыта, если передана из POI
- ✅ **Тип отзыва** - скрыт, если предустановлен как "poi_review"

### 2. Форма стала максимально простой:
Когда открывается из метки на карте, пользователь видит только:
- **Описание** (текстовое поле) - обязательно
- **Галочка для фото** - опционально
- Кнопки "Отмена" и "Создать отзыв"

### 3. Изменен заголовок:
- Было: "Создать отзыв"
- Стало: "Написать отзыв"

## Как это работает:

1. Пользователь кликает на метку на карте
2. Открывается модальное окно с информацией о POI
3. Пользователь нажимает "✍️ Оставить отзыв"
4. Открывается упрощенная форма с:
   - Только полем описания
   - Галочкой для фото
   - Кнопками отправки
5. Координаты, категория и тип отзыва заполняются автоматически

## Если форма открывается не из POI:

Если форма открывается не из метки (например, со страницы создания отзыва), то показываются все поля:
- Тип отзыва (poi_review/incident)
- Координаты
- Категория
- Описание
- Галочка для фото

## Готово! ✅

Теперь форма максимально простая и удобная для пользователей!
