# Frontend - Карта Здоровья

React приложение для модуля геймификации веб-приложения "Карта здоровья".

## Технологии

- **React 18** с TypeScript
- **React Router** для навигации
- **Styled Components** для стилизации
- **Framer Motion** для анимаций
- **Axios** для работы с API

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Укажите URL вашего Django backend в `.env`:

```
REACT_APP_API_URL=http://localhost:8000/api
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка для production

```bash
npm run build
```

## Структура проекта

```
src/
├── api/              # API клиенты и методы
├── components/       # React компоненты
│   ├── common/      # Переиспользуемые компоненты
│   └── layout/      # Компоненты макета
├── pages/           # Страницы приложения
├── styles/          # Глобальные стили
└── theme/           # Дизайн-система (цвета, типографика, отступы)
```

## Дизайн

Приложение использует современную дизайн-систему с:
- Яркими градиентами (бирюзовый/синий)
- Glassmorphism эффектами
- Плавными анимациями
- Игровыми элементами (прогресс-бары, значки, уровни)
